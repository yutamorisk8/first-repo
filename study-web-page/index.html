H1-H6要素

セクションの見出しを６段階で表す。

      
        <h1>これはh1の見出しです</h1>
        <h2>これはh2の見出しです</h2>

P要素

テキストの段落を表す。

      
        <p>これはテキストの最初の段落です。これはテキストの最初の段落です。</p>
        <p>これはテキストの二番目の段落です。これはテキストの二番目の段落です。</p>

IMG要素

文書に画像を埋め込む。

ul要素

項目の順序なしリストを表す。

      
        <img src="./assets/images/icon.png"alt="太郎のアイコン画像です">

li要素

リストの項目を表すために用いられる。

      
        <ul>
          <li>これは１個目のアイテム</li>
          <li>これは２個目のアイテム</li>
          <li>これは３個目のアイテム</li>
        </ul>

a要素

href 属性を用いて、別のウェブページ、ファイル、 メールアドレス、同一ページ内の場所、または他の URL へのハイパーリンクを作成する。 

      
        <a href="https://www.google.com""target="_blank">これはGoogleの検索ページへのリンクです。</a>
